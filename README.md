# 🚀 AI Resume Analyzer

An intelligent backend system that analyzes resumes using Django and NLP techniques to extract technical skills, evaluate experience level, and generate actionable improvement suggestions.

Built with a focus on real-world backend engineering, this project demonstrates file processing, natural language understanding, and structured decision-making logic — similar to systems used in modern hiring pipelines.

---

## 💡 Why This Project?

Most resume tools rely on manual review or expensive APIs. This system takes a different approach:

* Processes raw PDF resumes
* Extracts meaningful information using NLP
* Evaluates candidate strength using rule-based intelligence
* Generates structured feedback for improvement

---

## ⚙️ Core Capabilities

* 📄 PDF Resume Upload & Parsing
* 🧠 NLP-based Skill Extraction (spaCy)
* 📊 Resume Scoring Engine
* 🎯 Experience Level Classification
* 💡 Automated Improvement Suggestions
* 🎨 Clean UI with Bootstrap

---

## 🧠 Engineering Focus

This project is designed to demonstrate:

* Backend system design using Django
* File handling and document parsing
* NLP integration without external APIs
* Rule-based decision systems
* Clean separation of logic and presentation
